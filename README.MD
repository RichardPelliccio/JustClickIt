# Project Name

## Dependencies
- **Node.js**: Version `v20.17.0`

## Installation Instructions
1. Install required Node packages:
   ```sh
   npm install
   ```
2. Start the site:
   ```sh
   npm start
   ```

## Accessing the Site
Once the site is running, you can access it in your browser using any of the following:
- `http://localhost:80`
- `http://127.0.0.1:80`
- `http://[Your IPv4 Address]:80`
